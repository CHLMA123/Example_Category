Declared in NSLayoutConstraint+PureLayout.h

Overview

A category on NSLayoutConstraint that allows constraints to be easily installed & removed.

Other Methods

+ autoCreateAndInstallConstraints:

Creates all of the constraints in the block, then installs (activates) them all at once. All constraints created from calls to the PureLayout API in the block are returned in a single array. This may be more efficient than installing (activating) each constraint one-by-one.

+ (PL__NSArray_of ( NSLayoutConstraint *) *)autoCreateAndInstallConstraints:(__attribute__ ( ( noescape ) ) ALConstraintsBlock)block
